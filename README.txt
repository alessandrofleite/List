Exemplo de implementação da estrutura de dados Pilha em C.

O código está organizado da seguinte forma:
   list.h 			 		- arquivo de cabeçalho com a definição das funções inerente à estrutura de lista.
   list.c 			 		- implementação das funções definidas no arquivo de cabeçalho list.h
   ExemploUsoListaLinear.c 	- exemplo de uso da estrutura de dados do tipo Lista
   
   O programa de exemplo define uma pilha de 5.

Para compilar
  gcc list.c -o list
  gcc ExemploUsoListaLinear.c -o ExemploUsoListaLinear
  
Para executar
   ./ExemploUsoListaLinear
